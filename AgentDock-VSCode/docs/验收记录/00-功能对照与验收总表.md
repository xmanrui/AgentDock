# AgentDock VS Code 功能对照与验收总表

验收日期：2026-07-12

基准版本：同仓库 `agentdock-plugin` JetBrains 插件当前源码。

## 结论

JetBrains 版本当前 Dashboard 暴露的业务能力已全部移植到 VS Code：三家 Provider 的本地发现、筛选、卡片指标、聊天预览、套餐额度、Open/YOLO/Pin、终端存活点、Working/Ready/Idle、流式文本、退出识别、配置和缓存均有对应实现与测试。

唯一不能按像素位置 1:1 的部分是 JetBrains 对原生 Terminal Tab 的图标动画/角标和标题上方浮层。VS Code 公开 Extension API 不开放 Terminal Tab DOM 或动态标题图标绘制，因此改为在 AgentDock 会话卡片中显示相同状态与滚动文本。状态解析和生命周期逻辑没有缩减。

## 未移植为可见控件的能力

JetBrains 当前 Dashboard 本身没有显示 New、Rename、Archive 控件，README 也将其列为未开放能力。因此 VS Code 版本同样只提供当前可见的 Open、YOLO、Pin / Unpin，不把旧服务层中未开放的代码误算为用户功能。
